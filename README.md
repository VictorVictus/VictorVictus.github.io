Live at **[victorvictus.github.io](https://victorvictus.github.io)**

---

## Overview

A personal portfolio website built with pure HTML, CSS, and JavaScript. The site features a clean design aesthetic across multiple pages, including a professional CV and project showcases. Built with a focus on clean code and native browser APIs.

## Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
- **Dark Mode:** Supports automatic system preference and manual toggle (persisted via localStorage).
- **Internationalization:** Multi-language support (Catalan, Spanish, English, German).
- **Native Transitions:** Uses the View Transitions API for smooth navigation.
- **Performance:** Lightweight and dependency-free for fast loading.

The site is hosted on **GitHub Pages** and updated automatically from the `main` branch.
